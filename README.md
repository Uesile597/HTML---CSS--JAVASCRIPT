# Landing Page Fanta - Animacoes GSAP

Landing page responsiva da Fanta com animacoes incríveis no scroll usando GSAP + ScrollTrigger. Pratica de HTML5, CSS3 e JavaScript vanilla.

## Funcionalidades
- Hero animado: Garrafa Fanta, laranjas e folhas se movem sincronizadas no scroll
- Cards produtos: Coca-Cola, Fanta Laranja e Pepsi com limoes rotacionando  
- Navegacao fixa com menu hamburguer
- 100% responsivo (desktop/mobile)
- Animacoes suaves com GSAP timeline

## Tecnologias
HTML5 | CSS3 (Grid, Flexbox) | JavaScript Vanilla | GSAP 3.12.2 + ScrollTrigger | Font Product Sans
